## 本地使用

1. `git clone`
2. `npm install`
3. `bower install`
3. `gulp && make`  *总共运行了3条命令: 1. gulp  2. node app/app.js  3. gulp quickStart*
4. 浏览器打开 `http://localhost:8888/`( `browser-sync` ) 或者 `http://localhost:1337/`( `express` )
